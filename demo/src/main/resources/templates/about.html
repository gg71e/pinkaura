About Us

Where Your Glow Begins

Pink Aura is the result of my passion for combining beauty with technology. The idea for this project came from noticing the lack of specialized e-commerce platforms for beauty products in Egypt, similar to global experiences like Sephora in terms of ease of use and product variety.

I decided to take the initiative and build a complete e-commerce platform from scratch, handling all stages of development and design on my own.

I built the website structure and designed attractive, interactive interfaces using HTML, CSS, and Thymeleaf, focusing on providing a smooth and visually comfortable user experience.

I programmed all website features, from displaying products to managing the shopping cart, using JavaScript to ensure fast and responsive interaction.

I built a complete user management system including registration and login pages, with data security and validation, and connected everything to a MySQL database using Spring Boot JPA to ensure platform stability and flexibility.

Pink Aura is not just a graduation project or a school assignment. It is a reflection of my full-stack web development skills and my ability to turn a simple idea into a fully functional web application.

I built this project during long nights, with a lot of effort and dedication. I will continue to develop it and make it even stronger in the future.

This project is a story of challenge that proved to me that passion combined with hard work can build complex systems with full confidence.
